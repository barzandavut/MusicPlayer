
#  Music Player

## Teknolojiler

- **HTML5**
- **CSS3**
- **Bootstrap 5**
- **JavaScript (Vanilla)**

## Özellikler

🚀 Özellikler

Şarkı başlatma / durdurma

Sonraki / önceki şarkıya geçme

Ses kontrolü

Şarkı resmi, başlık ve sanatçı bilgisini gösterme

Toplam şarkı süresini ekranda gösterme

Progress bar (ilerleme çubuğu)
